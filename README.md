This converts not only chords (C, D-, E-, F,...) as described in original version of plugin.

# MuseScore to iReal Pro Export Plugin

This plugin is based in part on the MuseScore iRealPro plugin by Norman Schmidt, Copyright (c) 2017 Norman Schmidt.

Additional functionality and modifications Copyright (c) 2026 Ilvis Faulbaums.

Modifications include:

* iReal Pro chord-grid generation;
* MuseScore repeat-barline detection;
* time-signature export;
* automatic volta detection through MuseScore element navigation;
* HTML export for opening charts in iReal Pro.
